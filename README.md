## Stack

- **Frontend:** Vanilla HTML/CSS/JS — no framework bloat, instant load
- **Fonts:** Google Fonts (Instrument Serif + DM Sans / DM Mono)
- **Icons:** Inline SVG (Lucide-style)
- **Deploy:** Vercel (auto-deploy from main)
